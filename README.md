# Dashboard-PetShop

# Visão Geral  
Este repositório contém o dashboard desenvolvido em Power BI para visualizar e monitorar métricas de vendas e desempenho comercial. A visualização permite explorar indicadores como receita, quantidade vendida, margem, segmentação por produto/região e tendências ao longo do tempo.

# Funcionalidades Principais  
- Monitoramento de receita total e por período.  
- Comparativo de resultados mês a mês / ano a ano.  
- Análise de produtos mais vendidos e regiões com maior desempenho.  
- Visualização de margem de lucro e evolução ao longo do tempo.  
- Filtros interativos para segmentar por cliente, produto ou região.

# Tecnologias Utilizadas  
- Microsoft Power BI (desktop + web)  
- Modelo de dados com tabelas de vendas, produtos, clientes e tempo.  
- Transformações e limpagem de dados no Power Query.  
- Visualizações: gráficos de colunas, linhas, matrizes, cartões de KPI e filtros interativos.  
- Publicação online para acesso via web.
